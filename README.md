# Go Playground

This repository contains Go code examples and exercises for learning concurrency and parallelism in Go. My personal sandbox for mastering Golang. Features daily drills, implementation of Go idioms, and mini-services built while exploring the ecosystem.

## Directory Structure

- **learning/**: Contains the main Go module and examples.
  - **concurrency/**: Contains concurrency-related examples.

## Getting Started

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd golang-playground
   ```

2. Install dependencies:

   ```bash
   go mod tidy
   ```

3. Run the application:
   ```bash
   go run main.go
   ```
